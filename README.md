# Super Trunfo F1 (React Native)

## Como rodar

1. Instale o Expo CLI:
   npm install -g expo-cli

2. Crie um projeto:
   npx create-expo-app super-trunfo-f1

3. Substitua os arquivos gerados pelos deste projeto.

4. Rode:
   npm start

## Estrutura

- App.js
- src/components/PilotCard.js
- src/data/pilots.js
